# Step 7-C-io: Production host I/O Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Each task has a recommended **Model** tier (haiku / sonnet / opus). Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Harden `rust/kalico-host-rt/` into a production-correct multi-MCU host runtime by closing the Step-6 Plan-decision-C deferrals: production `MsgProtoParser`, NAK retransmit + RFC 6298 RTO, async event dispatch, identify-during-reconnect race recovery, `ArmError::QualityGate` detail, and per-MCU `request_id` correlation.

**Architecture:** Single-threaded poll-reactor per port, mirroring `klippy/chelper/serialqueue.c`. Public `KalicoHostIo` is `Send + Sync` with all mutable state inside the reactor thread. Callers submit via mpsc and block on a `sync_channel(1)` rendezvous. Six sequential phases (A–F), each producing a green build at end-of-phase.

**Tech Stack:** Rust 1.85, `serialport`, `flate2`, `serde_json`, `indexmap`, `arc-swap`. Mirrors Klipper's wire protocol bit-for-bit.

**Spec:** `docs/superpowers/specs/2026-04-30-step-7c-io-design.md` (1313 lines). Each task references the relevant spec section(s) — fetch them at task entry.

## Model tier guidance

Each task carries a `**Model:**` recommendation. Rules of thumb:

- **haiku** — purely mechanical: file move, Cargo.toml edit, single-line code change, running an existing command, verifying a hash. Steps that don't require holding much context.
- **sonnet** — most implementation: write code from a clear spec section, follow the TDD pattern, integrate well-defined components. About 80% of tasks.
- **opus** — genuine architectural reasoning: state-machine integration, race-condition analysis, multi-system debugging, novel logic. Reserve for the wire-protocol state machine in Phase C and the GC/race tasks.

If a sonnet/haiku task surfaces unexpected complexity during execution, it's fine to escalate to opus mid-task — but try the recommended tier first.

### A5: Create build-time invariant check script

**Model:** sonnet
**Spec:** §1.1 item 11
**Files:** Create `tools/check_async_event_registration.py`

- [ ] **Step 1:** Create the script.

```bash
mkdir -p tools
cat > tools/check_async_event_registration.py <<'EOF'
#!/usr/bin/env python3
"""
Build-time invariant: all kalico_* async event format strings must
register via _DECL_OUTPUT exclusively (no sendf-emitted kalico_* events).

Reads out/klipper.dict and asserts no msg name starting with 'kalico_'
appears in the 'responses' category UNLESS suffixed with '_response'
(those ARE responses, not async events).
"""
import json
import sys

ASYNC_EVENT_PREFIX = "kalico_"
RESPONSE_SUFFIX = "_response"

def main(dict_path: str) -> int:
    with open(dict_path) as f:
        d = json.load(f)

    violations = []
    for fmt in d.get("responses", {}):
        name = fmt.split()[0]
        if name.startswith(ASYNC_EVENT_PREFIX) and not name.endswith(RESPONSE_SUFFIX):
            violations.append(name)

    if violations:
        print("ERROR: kalico_* async events found in 'responses' category:", file=sys.stderr)
        for v in sorted(set(violations)):
            print(f"  - {v}", file=sys.stderr)
        print("\nAll kalico_* async events must use output(...) / _DECL_OUTPUT.", file=sys.stderr)
        return 1

    print(f"OK: no kalico_* async events miscategorized in {dict_path}")
    return 0

if __name__ == "__main__":
    sys.exit(main(sys.argv[1] if len(sys.argv) > 1 else "out/klipper.dict"))
EOF
chmod +x tools/check_async_event_registration.py
```


Expected: `OK: no kalico_* async events miscategorized in out/klipper.dict` (post-A4 regen).

### A15: Implement `EnumValue` Deserialize (single int OR `[start, count]`)

**Model:** sonnet
**Spec:** §4.2
**Files:** `rust/kalico-host-rt/src/host_io/parser.rs`

- [ ] **Step 1:** Append manual Deserialize impl.

```rust
impl<'de> serde::Deserialize<'de> for EnumValue {
    fn deserialize<D>(deserializer: D) -> Result<Self, D::Error>
    where D: serde::Deserializer<'de>,
    {
        use serde::de::Error;
        let value = serde_json::Value::deserialize(deserializer)?;
        if let Some(i) = value.as_i64() {
            return Ok(EnumValue::Single(i as i32));
        }
        if let Some(arr) = value.as_array() {
            if arr.len() == 2 {
                let start = arr[0].as_i64().ok_or_else(|| D::Error::custom("EnumValue range[0] not int"))? as i32;
                let count = arr[1].as_i64().ok_or_else(|| D::Error::custom("EnumValue range[1] not int"))? as i32;
                return Ok(EnumValue::Range { start, count });
            }
        }
        Err(D::Error::custom("EnumValue: expected int or [start, count]"))
    }
}
```

### A26: Implement `EnumTable::from_dict`

**Model:** sonnet
**Spec:** §4.5 (table builds from `EnumValue` enum)
**Files:** `rust/kalico-host-rt/src/host_io/parser.rs`

- [ ] **Step 1:** Append struct + impl.

```rust
use std::collections::HashMap;

#[derive(Debug, Clone)]
pub struct EnumTable {
    pub by_name: HashMap<String, i32>,
    pub by_int:  HashMap<i32, String>,
}

impl EnumTable {
    pub fn from_dict(d: &IndexMap<String, EnumValue>) -> Self {
        let mut by_name = HashMap::new();
        let mut by_int  = HashMap::new();
        for (name, value) in d {
            match value {
                EnumValue::Single(i) => {
                    by_name.insert(name.clone(), *i);
                    by_int.insert(*i, name.clone());
                }
                EnumValue::Range { start, count } => {
                    // Strip trailing digits from `name` to get `enum_root`.
                    // Mirrors klippy/msgproto.py:437-445.
                    let root: String = name.trim_end_matches(|c: char| c.is_ascii_digit()).to_string();
                    let prefix_num: i32 = name[root.len()..].parse().unwrap_or(0);
                    for i in 0..*count {
                        let key = format!("{}{}", root, prefix_num + i);
                        let val = start + i;
                        by_name.insert(key.clone(), val);
                        by_int.insert(val, key);
                    }
                }
            }
        }
        Self { by_name, by_int }
    }
}
```

```rust
pub fn apply_enumeration_wrapping(
    fields: Vec<(String, FieldType)>,
    enums: &IndexMap<String, IndexMap<String, EnumValue>>,
) -> Vec<(String, WrappedField)> {
    fields
        .into_iter()
        .map(|(field_name, ty)| {
            for enum_name in enums.keys() {
                if field_name == *enum_name
                    || field_name.ends_with(&format!("_{}", enum_name))
                {
                    return (
                        field_name,
                        WrappedField::Enumerated {
                            inner: ty,
                            enum_name: enum_name.clone(),
                        },
                    );
                }
            }
            (field_name, WrappedField::Plain(ty))
        })
        .collect()
}
```

### A33: Implement `encode_vlq` with OutOfRange check

**Model:** sonnet
**Spec:** §3.2 + §4.10
**Files:** `rust/kalico-host-rt/src/host_io/parser.rs`

- [ ] **Step 1:** Append function.

```rust
pub fn encode_vlq(out: &mut Vec<u8>, value: i64) -> Result<(), ParseError> {
    if !(i64::from(i32::MIN)..=i64::from(u32::MAX)).contains(&value) {
        return Err(ParseError::OutOfRange {
            value,
            range: "[i32::MIN, u32::MAX]",
        });
    }
    let mut v = value;
    if value < 0 {
        v += 1 << 32;
    }
    let mut bytes: [u8; 5] = [0; 5];
    let mut idx = 5usize;
    loop {
        idx -= 1;
        #[allow(clippy::cast_sign_loss, clippy::cast_possible_truncation)]
        {
            bytes[idx] = (v as u8) & 0x7F;
        }
        v >>= 7;
        if v == 0 || v == -1 { break; }
        if idx == 0 { break; }
    }
    let last = bytes.len() - 1;
    for b in &mut bytes[idx..last] {
        *b |= 0x80;
    }
    out.extend_from_slice(&bytes[idx..]);
    Ok(())
}
```

```rust
pub fn decode_vlq(buf: &[u8]) -> Result<(i64, usize), ParseError> {
    let mut value: i64 = 0;
    let mut consumed = 0;
    for &b in buf.iter().take(5) {
        consumed += 1;
        value = (value << 7) | i64::from(b & 0x7F);

            if (value & (1 << 31)) != 0 {
                value -= 1 << 32;
            }
            return Ok((value, consumed));
        }
```

```rust
        for format in dict.commands.keys().chain(dict.responses.keys()) {
            let name = format.split_whitespace().next().unwrap_or("").to_string();
            if !seen_msgnames.insert(name.clone()) {

        for (enum_name, table) in &dict.enumerations {
            enumerations.insert(enum_name.clone(), EnumTable::from_dict(table));
```

```rust
pub fn encode_field_value(out: &mut Vec<u8>, ty: FieldType, value: &FieldValue) -> Result<(), ParseError> {
    match (ty, value) {

        (FieldType::I32,   FieldValue::I32(v))  => encode_vlq(out, i64::from(*v)),
```

```rust
    pub fn encode(&self, cmd: &str) -> Result<Vec<u8>, ParseError> {
        let mut tokens = cmd.split_whitespace();
        let name = tokens.next().ok_or(ParseError::EmptyCommand)?;
        let spec = self.by_command_name.get(name)
            .ok_or_else(|| ParseError::UnknownCommand(name.to_string()))?;

        let mut provided: HashMap<&str, &str> = HashMap::new();
        for token in tokens {
            let (k, v) = token.split_once('=').ok_or(ParseError::MalformedArg)?;
            provided.insert(k, v);
```

### A62: Implement `decode_field_plain`

```rust
use crate::transport::MessageValue;

pub fn decode_field_plain(body: &[u8], ty: FieldType) -> Result<(MessageValue, usize), ParseError> {
    match ty {
        FieldType::U32 | FieldType::U16 | FieldType::Byte => {
            let (raw_i64, n) = decode_vlq(body)?;
            // Mask to u32 to match Python PT_uint32.parse's & 0xFFFFFFFF.
            Ok((MessageValue::U32(raw_i64 as u32), n))
        }
        FieldType::I32 | FieldType::I16 => {
            let (raw_i64, n) = decode_vlq(body)?;
            Ok((MessageValue::I32(raw_i64 as i32), n))
        }
        FieldType::String | FieldType::Buffer | FieldType::ProgmemBuffer => {
            // 1-byte length, then exactly that many raw bytes. NOT null-terminated.
            if body.is_empty() { return Err(ParseError::Truncated); }
            let len = body[0] as usize;
            if body.len() < 1 + len { return Err(ParseError::Truncated); }
            Ok((MessageValue::Bytes(body[1..=len].to_vec()), 1 + len))
        }
    }
}
```

```rust
        if packet.len() < MESSAGE_MIN { return Err(ParseError::ShortFrame); }

            DispatchSpec::Output(spec) => {
                let (name, params) = self.decode_output(&body[n..], spec)?;
                Ok(DecodedFrame::Output { name, params })
            }
        }
    }
```

```rust
fn python_repr_bytes(bytes: &[u8]) -> String {
    let mut out = String::from("b'");
    for &b in bytes {
        if b == b'\\' || b == b'\'' {
            out.push('\\');
            out.push(b as char);
        } else if (0x20..=0x7E).contains(&b) {
            out.push(b as char);
        } else {
            out.push_str(&format!("\\x{:02x}", b));
        }
    }
    out.push('\'');
    out
}

pub fn format_output_message(format: &str, values: &[MessageValue]) -> String {
    let mut out = String::new();
    let mut iter = format.chars().peekable();
    let mut value_idx = 0;
    while let Some(ch) = iter.next() {
        if ch != '%' {
            out.push(ch);
            continue;
        }
        let mut code = String::from("%");
        while let Some(&c) = iter.peek() {
            code.push(c);
            iter.next();
            if matches!(c, 'u' | 'i' | 'c' | 's') { break; }
        }
        if value_idx >= values.len() {
            out.push_str(&code);
            continue;
        }
        let v = &values[value_idx];
        value_idx += 1;
        match v {
            MessageValue::U32(n) => out.push_str(&n.to_string()),
            MessageValue::I32(n) => out.push_str(&n.to_string()),
            MessageValue::Bytes(b) => out.push_str(&python_repr_bytes(b)),
            MessageValue::String(s) => out.push_str(&format!("{:?}", s)),
            _ => out.push_str(&code),
        }
    }
    out
}
```

```rust
/// Run the Klipper identify handshake. Returns the loaded parser, the
/// final seq value seen on the wire, and any leftover rx_buf bytes the
/// caller's reactor / shim will inherit.
pub(crate) fn identify_handshake(
    port: &mut Box<dyn serialport::SerialPort>,
    timeout: std::time::Duration,
) -> Result<
    (crate::host_io::parser::MsgProtoParser, u8, Vec<u8>),
    crate::transport::TransportError,
> {
    let deadline = std::time::Instant::now() + timeout;
    let mut rx_buf: Vec<u8> = Vec::with_capacity(1024);
    let mut seq: u8 = 0;

    // Drain stale RX from any prior session (~300 ms cap).
    let drain_until = std::time::Instant::now() + std::time::Duration::from_millis(300);
    let mut scratch = [0u8; 4096];
    while std::time::Instant::now() < drain_until {
        port.set_timeout(std::time::Duration::from_millis(50))
            .map_err(|e| crate::transport::TransportError::Io(
                std::io::Error::other(format!("serialport: {e}"))
            ))?;
        match port.read(&mut scratch) {
            Ok(0) => break,
            Ok(_n) => {}
            Err(e) if e.kind() == std::io::ErrorKind::TimedOut => break,
            Err(e) => return Err(crate::transport::TransportError::Io(e)),
        }
    }

    // Issue identify in chunks (existing Step-6 stub logic).
    // The full body is lifted from the existing implementation:
    //   - hand-encoded `identify offset=N count=40` request
    //   - wait for `identify_response` with offset matching identify_data.len()
    //   - accumulate identify_data; break when chunk is empty
    //   - call MsgProtoParser::process_identify on the accumulated blob

    let mut identify_data: Vec<u8> = Vec::new();
    loop {
        // Use parser-less identify path: hand-roll the request encoding.
        let cmd = format!("identify offset={} count=40", identify_data.len());
        let mut payload = Vec::with_capacity(16);
        payload.push(0u8);    // msgid=0 → identify (DefaultMessages)
        let offset_arg: u32 = identify_data.len() as u32;
        let count_arg: u32 = 40;
        crate::host_io::parser::encode_vlq(&mut payload, i64::from(offset_arg))
            .expect("identify offset always in range");
        crate::host_io::parser::encode_vlq(&mut payload, i64::from(count_arg))
            .expect("identify count always in range");

        // Frame + write.
        let frame = crate::host_io::wire::build_frame(&payload, seq & 0x0F);
        port.write_all(&frame).map_err(crate::transport::TransportError::Io)?;
        port.flush().map_err(crate::transport::TransportError::Io)?;
        seq = seq.wrapping_add(1) & 0x0F;

        // Wait for identify_response.
        let attempt_deadline = deadline.min(
            std::time::Instant::now() + std::time::Duration::from_millis(150)
        );
        let resp = wait_for_identify_response(port, &mut rx_buf, attempt_deadline)?;
        let resp = resp.ok_or_else(|| crate::transport::TransportError::Parse(
            "identify timed out (Step-6 minimal shim, no NAK resync)".into(),
        ))?;

        let resp_offset = resp.0 as usize;    // (offset, data) tuple from wait_for_identify_response
        let chunk = resp.1;
        if resp_offset != identify_data.len() {
            // Stale response from drain window; retry.
            continue;
        }
        if chunk.is_empty() {
            break;
        }
        identify_data.extend_from_slice(&chunk);
        if std::time::Instant::now() >= deadline {
            return Err(crate::transport::TransportError::Parse(
                "identify exceeded 15s timeout".into(),
            ));
        }
        let _ = cmd;    // silence unused
    }

    // Parse the identify blob into a MsgProtoParser.
    let dict: crate::host_io::parser::DataDictionary =
        serde_json::from_slice(&identify_data)
            .map_err(|e| crate::transport::TransportError::Parse(format!("identify JSON: {e}")))?;
    let parser = crate::host_io::parser::MsgProtoParser::from_dictionary(dict)
        .map_err(|e| crate::transport::TransportError::Parse(format!("identify dict: {e:?}")))?;

    Ok((parser, seq, rx_buf))
}

/// Helper: wait for an identify_response packet from the port.
/// Returns Some((offset, data)) on success; None on timeout.
fn wait_for_identify_response(
    port: &mut Box<dyn serialport::SerialPort>,
    rx_buf: &mut Vec<u8>,
    deadline: std::time::Instant,
) -> Result<Option<(u32, Vec<u8>)>, crate::transport::TransportError> {
    let mut scratch = [0u8; 256];
    loop {
        let now = std::time::Instant::now();
        if now >= deadline { return Ok(None); }
        let remaining = deadline - now;
        let read_to = remaining.min(std::time::Duration::from_millis(100));
        port.set_timeout(read_to).map_err(|e| crate::transport::TransportError::Io(
            std::io::Error::other(format!("serialport: {e}"))
        ))?;
        match port.read(&mut scratch) {
            Ok(n) if n > 0 => {
                rx_buf.extend_from_slice(&scratch[..n]);
                while let Some(packet) = crate::host_io::wire::extract_packet(rx_buf) {
                    if let Some((offset, data)) = decode_identify_response(&packet) {
                        return Ok(Some((offset, data)));
                    }
                    // Pre-identify: drop other (NAK / unparseable) packets silently.
                }
            }
            Ok(_) => {}
            Err(e) if e.kind() == std::io::ErrorKind::TimedOut => {}
            Err(e) => return Err(crate::transport::TransportError::Io(e)),
        }
    }
}

/// Decode the wire `identify_response` (msg-id=0, payload = VLQ offset + length-prefixed bytes).
fn decode_identify_response(packet: &[u8]) -> Option<(u32, Vec<u8>)> {
    const MESSAGE_MIN: usize = 5;
    const MESSAGE_HEADER_SIZE: usize = 2;
    const MESSAGE_TRAILER_SIZE: usize = 3;
    if packet.len() < MESSAGE_MIN + 1 { return None; }
    let body = &packet[MESSAGE_HEADER_SIZE..packet.len() - MESSAGE_TRAILER_SIZE];
    if body.is_empty() { return None; }
    let mut pos = 0usize;
    let msgid = body[pos];
    pos += 1;
    if msgid != 0 { return None; }    // pre-identify only knows msg-id 0
    let (offset, n) = crate::host_io::parser::decode_vlq(&body[pos..]).ok()?;
    pos += n;
    if pos >= body.len() { return None; }
    let len = body[pos] as usize;
    pos += 1;
    if body.len() < pos + len { return None; }
    let data = body[pos..pos + len].to_vec();
    Some((offset as u32, data))
}
```

```rust
impl crate::transport::Transport for KalicoHostIo {
    fn call(
        &self,
        cmd: &str,
        expected_response_name: &str,
        timeout: std::time::Duration,
    ) -> Result<crate::transport::MessageParams, crate::transport::TransportError> {
        let mut shim = self.shim.lock().expect("shim mutex poisoned");

        // Encode + frame + write.
        let payload = shim.parser.encode(cmd)
            .map_err(|e| crate::transport::TransportError::Parse(format!("{e:?}")))?;
        let frame = build_frame(&payload, &mut shim.seq);
        shim.port.write_all(&frame).map_err(crate::transport::TransportError::Io)?;
        shim.port.flush().map_err(crate::transport::TransportError::Io)?;

        // Wait for response.
        let deadline = std::time::Instant::now() + timeout;
        loop {
            if let Some(idx) = shim.pending.iter().position(|(n, _)| n == expected_response_name) {
                let (_, params) = shim.pending.remove(idx).unwrap();
                return Ok(params);
            }
            let now = std::time::Instant::now();
            if now >= deadline { return Err(crate::transport::TransportError::Timeout); }
            pump_rx(&mut shim, deadline - now)?;
        }
    }

// build_frame is already in wire.rs from C5. pump_rx is the shim-side helper that
// drains the port and parses packets into the shim's pending queue.
fn pump_rx(shim: &mut KalicoHostIoShimInner, timeout: std::time::Duration)
    -> Result<(), crate::transport::TransportError>
{
    let mut scratch = [0u8; 256];
    let read_to = timeout.min(std::time::Duration::from_millis(100));
    shim.port.set_timeout(read_to)
        .map_err(|e| crate::transport::TransportError::Io(
            std::io::Error::other(format!("serialport: {e}"))
        ))?;
    match shim.port.read(&mut scratch) {
        Ok(n) if n > 0 => {
            shim.rx_buf.extend_from_slice(&scratch[..n]);
            while let Some(packet) = crate::host_io::wire::extract_packet(&mut shim.rx_buf) {
                if packet.len() >= 2 {
                    shim.seq = packet[1] & 0x0F;
                }
                if let Ok(decoded) = shim.parser.decode(&packet) {
                    match decoded {
                        crate::host_io::parser::DecodedFrame::Response { name, params } => {
                            shim.pending.push_back((name, params));
                        }
                        crate::host_io::parser::DecodedFrame::Output { name, params } => {
                            shim.pending.push_back((name, params));
                        }
                    }
                }
            }
            Ok(())
        }
        Ok(_) => Ok(()),
        Err(e) if e.kind() == std::io::ErrorKind::TimedOut => Ok(()),
        Err(e) if e.kind() == std::io::ErrorKind::Interrupted => Ok(()),
        Err(e) if e.kind() == std::io::ErrorKind::WouldBlock => Ok(()),
        Err(e) => Err(crate::transport::TransportError::Io(e)),
    }
}
```

```rust
fn secs_mul(d: Duration, f: f64) -> Duration {
    Duration::from_secs_f64(d.as_secs_f64() * f)
}

fn clamp(d: Duration, min: Duration, max: Duration) -> Duration {
    if d < min { min } else if d > max { max } else { d }
}

impl RttEstimator {
    pub fn update(&mut self, r: Duration) {
        match self.srtt {
            None => {
                self.srtt = Some(r);
                self.rttvar = Some(r / 2);
            }
            Some(srtt) => {
                let diff = if srtt > r { srtt - r } else { r - srtt };
                let rttvar_new = secs_mul(self.rttvar.unwrap(), 1.0 - BETA) + secs_mul(diff, BETA);
                self.rttvar = Some(rttvar_new);
                self.srtt = Some(secs_mul(srtt, 1.0 - ALPHA) + secs_mul(r, ALPHA));
            }
        }
        let rttvar = self.rttvar.unwrap();
        let k_rttvar = secs_mul(rttvar, K);
        let rto_raw = self.srtt.unwrap() + std::cmp::max(G, k_rttvar);
        self.rto = clamp(rto_raw, MIN_RTO, MAX_RTO);
    }
}
```

```rust
//! Wire framing. Spec §3.4.

const MESSAGE_MIN: usize = 5;
const MESSAGE_DEST: u8 = 0x10;
const MESSAGE_SEQ_MASK: u8 = 0x0F;
pub const MESSAGE_SYNC: u8 = 0x7E;
const MESSAGE_PAYLOAD_MAX: usize = 64 - MESSAGE_MIN;

/// CRC16-CCITT over the header + payload. Klipper variant; mirrors
/// klippy/msgproto.crc16_ccitt byte-for-byte.
pub fn crc16_ccitt(buf: &[u8]) -> u16 {
    let mut crc: u16 = 0xFFFF;
    for &byte in buf {
        let data = u16::from(byte) ^ (crc & 0x00FF);
        let data = (data ^ ((data << 4) & 0x00FF)) & 0xFF;
        crc = (crc >> 8) ^ (data << 8) ^ (data << 3) ^ (data >> 4);
    }
    crc
}

/// Build a single wire frame from a payload + seq nibble.
/// Frame layout: [len, dest|seq, payload..., crc_hi, crc_lo, SYNC].
pub fn build_frame(payload: &[u8], wire_seq_nibble: u8) -> Vec<u8> {
    debug_assert!(payload.len() <= MESSAGE_PAYLOAD_MAX);
    let msglen = MESSAGE_MIN + payload.len();
    let seq_byte = (wire_seq_nibble & MESSAGE_SEQ_MASK) | MESSAGE_DEST;
    let mut frame = Vec::with_capacity(msglen);
    frame.push(msglen as u8);
    frame.push(seq_byte);
    frame.extend_from_slice(payload);
    let crc = crc16_ccitt(&frame);
    frame.push((crc >> 8) as u8);
    frame.push((crc & 0xFF) as u8);
    frame.push(MESSAGE_SYNC);
    frame
}
```

```rust
/// Extract a single packet from the rx buffer, draining bytes consumed.
/// Returns None if no complete packet is available; resyncs past garbage.
pub fn extract_packet(buf: &mut Vec<u8>) -> Option<Vec<u8>> {
    while !buf.is_empty() {
        let msglen = buf[0] as usize;
        if !(MESSAGE_MIN..=64).contains(&msglen) {
            buf.remove(0);
            continue;
        }
        if buf.len() < msglen { return None; }
        let seq_byte = buf[1];
        if (seq_byte & !MESSAGE_SEQ_MASK) != MESSAGE_DEST || buf[msglen - 1] != MESSAGE_SYNC {
            buf.remove(0);
            continue;
        }
        let crc_off = msglen - 3;
        let crc_expected = (u16::from(buf[crc_off]) << 8) | u16::from(buf[crc_off + 1]);
        let crc_actual = crc16_ccitt(&buf[..crc_off]);
        if crc_expected != crc_actual {
            buf.remove(0);
            continue;
        }
        let pkt = buf[..msglen].to_vec();
        buf.drain(..msglen);
        return Some(pkt);
    }
    None
}
```

```rust
//! UnackedWindow + AwaitingResponse. Spec §3.3.

use std::collections::VecDeque;
use std::time::Instant;

pub const MAX_PENDING_BLOCKS: usize = 12;

#[derive(Debug)]
pub struct UnackedEntry {
    pub seq:         u64,
    pub frame_bytes: Vec<u8>,
    pub sent_at:     Instant,
    pub retry_count: u32,
}

#[derive(Debug, Default)]
pub struct UnackedWindow {
    entries: VecDeque<UnackedEntry>,
}

impl UnackedWindow {
    pub fn len(&self) -> usize { self.entries.len() }
    pub fn is_empty(&self) -> bool { self.entries.is_empty() }
    pub fn is_full(&self) -> bool { self.entries.len() >= MAX_PENDING_BLOCKS }
    pub fn front(&self) -> Option<&UnackedEntry> { self.entries.front() }

    pub fn push(&mut self, entry: UnackedEntry) {
        debug_assert!(!self.is_full(), "UnackedWindow overflow");
        self.entries.push_back(entry);
    }

    /// Pop entries with seq < rseq (strict; per Codex finding #2).
    /// Returns the popped entries for RTT-sample post-processing.
    pub fn pop_acked(&mut self, rseq: u64) -> Vec<UnackedEntry> {
        let mut popped = Vec::new();
        while let Some(front) = self.entries.front() {
            if front.seq < rseq {
                popped.push(self.entries.pop_front().unwrap());
            } else {
                break;
            }
        }
        popped
    }

    pub fn iter(&self) -> impl Iterator<Item = &UnackedEntry> {
        self.entries.iter()
    }

    pub fn iter_mut(&mut self) -> impl Iterator<Item = &mut UnackedEntry> {
        self.entries.iter_mut()
    }
}
```

```rust
use std::sync::mpsc::SyncSender;
use crate::transport::{MessageParams, TransportError};

#[derive(Debug)]
pub struct AwaitEntry {
    pub call_id:                u64,
    pub seq:                    u64,
    pub expected_response_name: String,
    pub completion:             SyncSender<Result<MessageParams, TransportError>>,
    pub submitted_at:           Instant,
    pub deadline:               Instant,
    pub abandoned:              bool,
}

#[derive(Debug, Default)]
pub struct AwaitingResponse {
    entries: VecDeque<AwaitEntry>,
}

const AWAITING_DEFENSIVE_CEILING: usize = 1024;

impl AwaitingResponse {
    pub fn len(&self) -> usize { self.entries.len() }

    pub fn push(&mut self, entry: AwaitEntry) -> Result<(), TransportError> {
        if self.entries.len() >= AWAITING_DEFENSIVE_CEILING {
            return Err(TransportError::Parse("AwaitingResponse defensive ceiling exceeded".into()));
        }
        self.entries.push_back(entry);
        Ok(())
    }

    /// FIFO match against expected_response_name, skipping abandoned entries.
    /// Returns the matched entry's index for caller-side removal.
    pub fn find_match(&self, name: &str) -> Option<usize> {
        self.entries.iter().position(|e| !e.abandoned && e.expected_response_name == name)
    }

    pub fn remove(&mut self, idx: usize) -> AwaitEntry {
        self.entries.remove(idx).expect("idx valid")
```

```rust
use crate::host_io::window::UnackedEntry;

const PENDING_SUBMISSION_CEILING: usize = 256;

impl Reactor {
    fn dispatch_submission(
        &mut self,
        call_id: u64,
        payload: Vec<u8>,
        expected_response_name: String,
        completion: std::sync::mpsc::SyncSender<Result<MessageParams, TransportError>>,
        deadline: std::time::Instant,
    ) -> Result<(), TransportError> {
        if self.unacked_window.is_full() {
            // Defer per spec §3.4 step 1. Try-drain happens in drain_pending_submissions
            // every reactor loop iteration after an ack frees a slot.
            if self.pending_submissions.len() >= PENDING_SUBMISSION_CEILING {
                let _ = completion.send(Err(TransportError::Parse(
                    "pending submission queue overflow".into(),
                )));
                return Ok(());
            }
            self.pending_submissions.push_back(PendingSubmission {
                call_id, payload, expected_response_name, completion, deadline,
            });
            return Ok(());
        }

        let seq = self.send_seq;
        self.send_seq += 1;
        let wire_seq = (seq & 0x0F) as u8;
        let frame = build_frame(&payload, wire_seq);

        self.write_frame(&frame)?;

        let now = std::time::Instant::now();
        self.unacked_window.push(UnackedEntry {
            seq, frame_bytes: frame.clone(), sent_at: now, retry_count: 0,
        });
        let entry = crate::host_io::window::AwaitEntry {
            call_id, seq,
            expected_response_name,
            completion,
            submitted_at: now,
            deadline,
            abandoned: false,
        };
        self.awaiting_response.push(entry)?;

        // RTT-sample arming (§3.4 step 6).
        if !self.rtt_sample_armed {
            self.rtt_sample_seq = seq;
            self.rtt_sample_armed = true;
        }
        Ok(())
    }

    /// Try to drain the pending submission queue if the window has room.
    /// Called from the reactor loop after each ack potentially frees a slot.
    pub(crate) fn drain_pending_submissions(&mut self) {
        while !self.unacked_window.is_full() {
            let Some(p) = self.pending_submissions.pop_front() else { break; };
            // Re-dispatch (recursively safe — the window check at the top of
            // dispatch_submission would just re-enqueue, but we just popped a
            // slot so the window has room).
            let _ = self.dispatch_submission(
                p.call_id, p.payload, p.expected_response_name, p.completion, p.deadline,
            );
        }
    }
}
```

```rust
const MESSAGE_SEQ_MASK: u64 = 0x0F;

impl Reactor {
    /// Decode wire seq nibble against current receive_seq.
    fn decode_absolute(&self, wire_seq: u8) -> u64 {
        let delta = (u64::from(wire_seq) & MESSAGE_SEQ_MASK).wrapping_sub(self.receive_seq) & MESSAGE_SEQ_MASK;
        // Note: the `(- receive_seq)` mod 16 is the correct delta; absolute is in [receive_seq, receive_seq+15].
        let delta = (u64::from(wire_seq).wrapping_sub(self.receive_seq)) & MESSAGE_SEQ_MASK;
        self.receive_seq + delta
    }

    /// Inbound 5-byte ack/nak handler. Per spec §3.5 (corrected per Codex finding #1).
    fn handle_ack_nak(&mut self, wire_seq_nibble: u8) -> Result<(), TransportError> {
        let rseq = self.decode_absolute(wire_seq_nibble);

        // Step 1: advance receive_seq if rseq is new.
        if rseq > self.receive_seq {
            self.update_receive_seq(rseq)?;
        }

        // Step 2: ack/nak handling — runs in BOTH cases (forward progress AND duplicate ack).
        if self.last_ack_seq < rseq {
            self.last_ack_seq = rseq;
        } else if rseq > self.ignore_nak_seq && !self.unacked_window.is_empty() {
            // Duplicate-ack-as-NAK → fast retransmit, INLINE.
            self.write_retransmit(RetransmitTrigger::NakDriven)?;
        }
        // else: stale ack damped by ignore_nak_seq, drop silently.
        Ok(())
    }

    /// Advance receive_seq + pop UnackedWindow entries with seq < rseq.
    /// First-connection sentinel (§3.10): if window is empty, snap send_seq + receive_seq to rseq.
    fn update_receive_seq(&mut self, rseq: u64) -> Result<(), TransportError> {
        if self.unacked_window.is_empty() {
            // Connection-init / mid-session-reopen short-circuit.
            self.send_seq = rseq;
            self.receive_seq = rseq;
            return Ok(());
        }

        let popped = self.unacked_window.pop_acked(rseq);
        // RTT sample on the first popped entry whose seq >= rtt_sample_seq.
        for entry in &popped {
            if entry.seq >= self.rtt_sample_seq && self.rtt_sample_armed {
                let rtt = std::time::Instant::now() - entry.sent_at;
                self.rtt.update(rtt);
                self.rtt_sample_armed = false;
                break;
            }
        }
        self.receive_seq = rseq;
        Ok(())
    }
}
```

### C20: Implement `Reactor::write_retransmit` (two-arm `ignore_nak_seq` per Codex finding #7)

**Model:** opus
**Spec:** §3.8

```rust
const MAX_RETRY_COUNT: u32 = 8;

impl Reactor {
    fn write_retransmit(&mut self, trigger: RetransmitTrigger) -> Result<(), TransportError> {
        // Build retransmit buffer.
        let frames: Vec<&[u8]> = self.unacked_window.iter()
            .map(|e| e.frame_bytes.as_slice())
            .collect();
        let buf = crate::host_io::wire::build_retransmit_buffer(frames);
        self.write_frame(&buf)?;

        // Two-arm ignore_nak_seq (Codex finding #7).
        match trigger {
            RetransmitTrigger::NakDriven => {
                if self.receive_seq < self.retransmit_seq {
                    // Second NAK before receiver caught up — bump damper.
                    self.ignore_nak_seq = self.retransmit_seq;
                } else {
                    self.ignore_nak_seq = self.receive_seq;
                }
            }
            RetransmitTrigger::TimeoutDriven => {
                self.ignore_nak_seq = self.send_seq;
            }
        }

        self.retransmit_seq = self.send_seq;
        self.rtt_sample_armed = false;

        // Increment retry counts; if any hits cap, fault.
        for entry in self.unacked_window.iter_mut() {
            entry.retry_count += 1;
            if entry.retry_count >= MAX_RETRY_COUNT {
                self.state = ReactorState::Closed;
                self.pending_host_fault = Some(crate::host_io::runtime_events::FaultEvent {
                    fault_code:   runtime::error::KALICO_ERR_HOST_RETRANSMIT_EXHAUSTED as u16,
                    fault_detail: entry.retry_count,
                    segment_id:   0,    // retransmit-exhausted is wire-level, not segment-scoped
                    synthesized:  false,
                });
                return Err(TransportError::Closed);
            }
        }

        // RTO backoff ONLY on TimeoutDriven (Codex finding #7).
        if matches!(trigger, RetransmitTrigger::TimeoutDriven) {
            self.rtt.backoff();
        }
        Ok(())
    }
}
```

```rust
impl Reactor {
    fn handle_inbound_frame(&mut self, packet: Vec<u8>) -> Result<(), TransportError> {
        const MESSAGE_MIN: usize = 5;
        if packet.len() < MESSAGE_MIN { return Ok(()); }

        let wire_seq_nibble = packet[1] & 0x0F;
        if packet.len() == MESSAGE_MIN {
            // 5-byte ack/nak frame.
            self.handle_ack_nak(wire_seq_nibble)?;
            return Ok(());

        let rseq = self.decode_absolute(wire_seq_nibble);
        if rseq != self.receive_seq {
            self.update_receive_seq(rseq)?;

        // and the frame is dropped (NOT propagated as Closed). Dictionary
        // version skew, unknown msg-ids, etc. are recoverable.
        let decoded = match self.parser.decode(&packet) {
            Ok(d) => d,
            Err(e) => {

                return Ok(());

            DecodedFrame::Output { name, params } => {
                let event = RuntimeEvent::lift(&name, params);
                self.dispatch_runtime_event(event);
            }
        }
        Ok(())
```

### C22: Implement `Reactor::poll_loop` step

**Model:** opus
**Spec:** §3.7

- [ ] **Step 1:** Append the loop.

```rust
const MAX_SUBMITS_PER_ITER: usize = 4;
const READ_TIMEOUT: Duration = Duration::from_millis(100);

impl Reactor {
    pub fn run(&mut self) {
        loop {
            // 1. Drain reactor commands (bounded per iteration).
            for _ in 0..MAX_SUBMITS_PER_ITER {
                match self.submission_rx.try_recv() {
                    Ok(cmd) => self.handle_command(cmd),
                    Err(std::sync::mpsc::TryRecvError::Empty) => break,
                    Err(std::sync::mpsc::TryRecvError::Disconnected) => {
                        self.state = ReactorState::Closed;
                        break;
                    }
                }
            }

            // 2. Poll serial port.
            self.poll_serial();

            // 3. RTO timer step.
            if let Some(front) = self.unacked_window.front() {
                let now = Instant::now();
                if now >= front.sent_at + self.rtt.current_rto() {
                    let _ = self.write_retransmit(RetransmitTrigger::TimeoutDriven);
                }
            }

            // 4. AwaitingResponse GC (layer 2 — per-entry deadline).
            let now = Instant::now();
            let evicted = self.awaiting_response.evict_expired(now);
            for entry in evicted {
                let _ = entry.completion.send(Err(TransportError::DispatcherTimeout));
            }

            // 5. Closed-state exit.
            if self.state == ReactorState::Closed {
                self.flush_all_completions();
                break;
            }
        }
    }

    fn handle_command(&mut self, cmd: ReactorCommand) {
        match cmd {
            ReactorCommand::Submit { call_id, cmd, expected_response_name, completion, deadline } => {
                // Encode + dispatch.
                match self.parser.encode(&cmd) {
                    Ok(payload) => {
                        if let Err(e) = self.dispatch_submission(call_id, payload, expected_response_name, completion.clone(), deadline) {
                            let _ = completion.send(Err(e));
                        }
                    }
                    Err(e) => {
                        let _ = completion.send(Err(TransportError::Parse(format!("{e:?}"))));
                    }
                }
            }
            ReactorCommand::SubmitTyped { call_id, payload, expected_response_name, completion, deadline } => {
                let _ = self.dispatch_submission(call_id, payload, expected_response_name, completion, deadline);
            }
            ReactorCommand::Abandon(call_id) => {
                self.awaiting_response.mark_abandoned(call_id);
            }
            ReactorCommand::Shutdown => {
                self.state = ReactorState::Closed;
            }
            // Subscribe* variants: handled by EventDispatcher (Phase D); for now, send Closed.
            ReactorCommand::AttachCreditCounter(_)
            | ReactorCommand::SubscribeFault { .. }
            | ReactorCommand::SubscribeTrace { .. }
            | ReactorCommand::SubscribeRuntimeEvents { .. }
            | ReactorCommand::SubscribeHostEvents { .. } => {
                // Phase-D wiring TBD.
            }
        }
    }

    fn poll_serial(&mut self) {
        let mut scratch = [0u8; 256];
        if self.port.set_timeout(READ_TIMEOUT).is_err() { return; }
        match self.port.read(&mut scratch) {
            Ok(n) if n > 0 => {
                self.rx_buf.extend_from_slice(&scratch[..n]);
                while let Some(packet) = crate::host_io::wire::extract_packet(&mut self.rx_buf) {
                    let _ = self.handle_inbound_frame(packet);
                }
            }
            Ok(_) => {}
            Err(e) if e.kind() == std::io::ErrorKind::TimedOut => {}
            Err(e) if e.kind() == std::io::ErrorKind::Interrupted => {}
            Err(e) if e.kind() == std::io::ErrorKind::WouldBlock => {}
            Err(e) => {
                // Non-recoverable; transition closed and stage the host fault.
                // FaultLatch dispatch wired in Phase D Task D11b once
                // EventDispatcher is composed onto the reactor.
                log::warn!("port read error: {e:?}; transitioning to Closed");
                self.pending_host_fault = Some(crate::host_io::runtime_events::FaultEvent {
                    fault_code:   runtime::error::KALICO_ERR_HOST_DISCONNECT as u16,
                    fault_detail: 0,
                    segment_id:   0,
                    synthesized:  false,
                });
                self.state = ReactorState::Closed;
            }
        }
    }

    fn flush_all_completions(&mut self) {
        // Layer 3 GC: disconnect-clears-all.
        while let Some(idx) = self.awaiting_response.find_match(/* any */ "") {
            let entry = self.awaiting_response.remove(idx);
            let _ = entry.completion.send(Err(TransportError::Closed));
        }
        // (Note: find_match("") won't match anything; this loop is a placeholder.
        // Real impl needs a `drain_all()` method on AwaitingResponse.)
    }
}
```

**Files:** `rust/kalico-host-rt/src/host_io/events.rs`

```rust
use std::sync::mpsc::{SyncSender, TrySendError};
use crate::host_io::runtime_events::TraceEvent;

#[derive(Debug)]
pub struct TraceRing {
    capacity:                  usize,
    sticky_overflow:           bool,
    subscriber:                Option<SyncSender<TraceEvent>>,
    drop_count_since_event:    u64,
    host_event_tx:             Option<SyncSender<HostEvent>>,
}

impl TraceRing {
    pub fn new(capacity: usize) -> Self {
        Self {
            capacity,
            sticky_overflow: false,
            subscriber: None,
            drop_count_since_event: 0,
            host_event_tx: None,
        }
    }

    pub fn dispatch(&mut self, mut event: TraceEvent) {
        if self.sticky_overflow {
            event.flags |= 0x01;    // OVERFLOW flag
            self.sticky_overflow = false;
        }

        match self.subscriber.as_ref() {
            Some(tx) => match tx.try_send(event) {
                Ok(_) => {}
                Err(TrySendError::Full(_)) => {
                    self.sticky_overflow = true;
                    self.drop_count_since_event += 1;
                    if let Some(host_tx) = &self.host_event_tx {
                        let _ = host_tx.try_send(HostEvent::TraceSubscriberOverflow {
                            dropped_count: self.drop_count_since_event,
                            at: Instant::now(),
                        });
                    }
                }
                Err(TrySendError::Disconnected(_)) => {
                    self.subscriber = None;
                    self.drop_count_since_event = 0;
                    if let Some(host_tx) = &self.host_event_tx {
                        let _ = host_tx.try_send(HostEvent::TraceSubscriberDisconnected {
                            at: Instant::now(),
                        });
                    }
                }
            },
            None => self.drop_count_since_event += 1,
        }
    }

    pub fn subscribe(&mut self, tx: SyncSender<TraceEvent>) -> Result<(), crate::transport::SubscribeError> {
        if self.subscriber.is_some() {
            return Err(crate::transport::SubscribeError::AlreadySubscribed { channel: "trace" });
        }
        // Re-armable: post-disconnect, subscribe is allowed.
        if self.drop_count_since_event > 0 {
            if let Some(host_tx) = &self.host_event_tx {
                let _ = host_tx.try_send(HostEvent::TraceSubscriberReattached {
                    events_lost_during_gap: self.drop_count_since_event,
                    at: Instant::now(),
                });
            }
            self.drop_count_since_event = 0;
        }
        self.subscriber = Some(tx);
        Ok(())
    }

    pub fn set_host_event_tx(&mut self, tx: SyncSender<HostEvent>) {
        self.host_event_tx = Some(tx);
    }
}
```
